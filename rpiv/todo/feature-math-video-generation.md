# 智能数学讲解视频生成系统

## 动机与背景

为两个孩子（初三 + 高一）提供极致质量的数学讲解。拍照上传题目（含图形和文字），自动生成带有几何动画、高亮、详细步骤和定理说明的讲解视频。仅私人使用，不计成本追求最优效果。

## 期望行为

- 拍照/截图上传一道数学题（支持几何图形 + 文字混合）
- 系统自动识别题目内容，生成结构化解题步骤
- 输出 1080P/60FPS 的 Manim 动画讲解视频，配合 TTS 语音讲解
- 每一步推导与动画、语音精准对齐
- 数学推导经过符号计算引擎验证，确保零偏差

## 用户场景

1. **初三几何压轴题**：上传含手绘辅助线的平面几何题，生成带动点轨迹动画的讲解视频
2. **高一立体几何**：上传空间几何体题目，生成可旋转视角的 3D 动画，将立体问题降维展示
3. **函数与解析式**：上传二次函数/抽象函数题，生成函数图像动态绘制 + 关键点标注的视频

## MVP 定义

核心管线（Agentic Workflow）四个阶段：

1. **多模态数学解构**：上传照片 → Claude Sonnet 4.6 输出结构化 JSON（LaTeX + 已知条件 + 解题步骤）
2. **剧本与代码编译**：解题步骤 + Manim System Prompt → 生成 Python 动画代码（禁止硬编码坐标，必须用几何约束计算）
3. **沙盒执行与自愈循环**：本地执行 Manim CLI → 报错自动送回 Claude 修复（最多 3 次重试，目标成功率 95%+）
4. **音视频合成**：TTS 生成语音 → FFmpeg 混音 → 输出 MP4

### 三级交叉验证架构

- **第一级**：SymPy 前置求解，获取数学真值作为硬约束注入 Prompt
- **第二级**：Manim 代码中插入 assert 断言（如向量点积验证垂直关系），失败则熔断重试
- **第三级（可选）**：Lean 4 形式化定理证明器，用于竞赛级高难度题

### 技术栈

| 组件 | 选型 |
|------|------|
| AI 引擎 | Claude Sonnet 4.6（100 万 token 上下文） |
| 动画渲染 | Manim Community Edition |
| 符号计算 | SymPy |
| 语音合成 | ElevenLabs（极致质量）/ Azure Neural TTS（备选） |
| 环境管理 | uv |
| 音视频处理 | FFmpeg |
| 部署 | 本地设备（渲染成本归零） |

## 备选方案

- **前端原生渲染路线**：后端返回 JSON 动画指令 + TTS 数据，客户端用 JSXGraph/GeoGebra Web 本地渲染。延迟低、成本低（¥0.20/题），但视觉效果和交互体验不如 Manim 视频方案
- 如果后续需要商业化扩展，可切换为此路线
