## 需求摘要

### 产品愿景
- **核心问题**：初高中学生日常学习中遇到难题缺少即时、高质量、零偏差的讲解辅导
- **价值主张**：AI 驱动的个性化学习助手，提供经过符号验证的解题辅导和多维度学情诊断
- **目标用户**：初中和高中学生（不限年级），当前重点服务初二和初三
- **初始用户**：约 3 人（自己的孩子初二 + 两位同事的孩子初三）
- **产品形态**：Web 应用（浏览器访问，兼顾手机拍照和电脑使用）
- **部署方式**：Railway 云端部署，所有用户均可远程访问

### 核心场景（按优先级排序）
1. **智能解题**：拍照上传 → AI 解析 → 经验证的文字解题步骤（苏格拉底式引导 / 完整解答可切换，支持追问）
2. **学情诊断**：试卷上传 + 解题记录 → 多维度错因分析 → 诊断报告 + 从题库推荐针对性练习题
3. **用户系统**：多用户注册 + 年级 profile + 解题历史 + 学情数据隔离 + 基于年级的个性化适配

### 产品边界

**MVP 范围内**：
- 智能解题（文字双模式：苏格拉底引导/完整解答，拍照/截图输入，多轮追问）
- 学科策略分层：
  - **数学**（深度模式）：三级交叉验证（SymPy 前置求解 + assert 断言 + 可选 Lean 4）+ 结构化题库
  - **理化生**（基础模式）：纯多模态 LLM 解题，无符号验证，依赖模型能力
- 学情诊断（多维错因分析 + 弱项报告 + 练习题推荐，从试卷上传快速冷启动）
- 用户系统（多用户注册、年级 profile、解题历史）
- 题库（数学：CMM-Math/TAL-SCQ5K-CN 打底 + 日常作业积累 + LLM 生成变式题）
- 轻量个性化（根据年级 + 学科自动调整讲解深度）

**明确不做（第一版）**：
- Manim 视频生成
- 语音交互（TTS / 语音输入）
- 学习路径规划
- 移动端原生 App

**后续版本考虑**：
- Manim 动画讲解视频（Math-To-Manim Pipeline）
- 语音实时互动（参考 Gauth AI Live Tutor）
- 遗忘曲线驱动的复习推荐
- 知识图谱可视化
- 学习路径规划
- 理化生学科的验证机制和专项题库

### 已知约束
- 初始用户量极小（约 3 人），不需考虑高并发
- 不计成本追求质量
- Railway 云端部署
- 未来可能扩展到更多用户，架构上不做硬限制

### 各场景功能要点

#### 场景 1：智能解题
- **输入**：拍照/截图上传（多模态 LLM 端到端识别，Pix2Text 作为补充）
- **输出**：结构化解题步骤（LaTeX 公式 + 文字说明 + 定理/知识点标注）
- **交互**：默认苏格拉底式引导（参考 Khanmigo/Mr. Ranedeer 的 Prompt 设计），可切换完整解答；支持多轮追问
- **验证**（数学学科）：三级交叉验证
  - 第一级：SymPy 前置求解，获取数学真值作为硬约束注入 Prompt
  - 第二级：关键步骤 assert 断言（如向量点积验证垂直关系），失败触发自动重试
  - 第三级（可选）：Lean 4 形式化定理证明，用于高难度题
- **验证**（理化生学科）：依赖多模态 LLM 自身推理能力，无额外符号验证
- **个性化**：根据用户年级自动调整讲解深度和知识点引用范围
- **异常处理**：图片模糊提示重新上传；AI 不确定时标注置信度；验证失败自动重试（最多 3 次）

#### 场景 2：学情诊断
- **冷启动策略**：让孩子上传近期试卷，快速积累初始数据；数据不足阶段用 LLM 做基础分析，数据充足后切换到 KT 模型
- **输入**：试卷拍照上传 + 系统内解题记录
- **对错判定**：AI 自动解题比对 + 孩子可手动修正
- **诊断维度**（参考小猿五重错因分析）：
  1. 知识漏洞检测：缺失哪些知识点
  2. 思维路径回溯：解题思路在哪一步断裂
  3. 概念关联分析：相关联的知识点是否掌握
  4. 解题习惯诊断：是否存在系统性的解题偏差
  5. 认知水平评估：当前处于哪个认知层级
- **输出**：多维度诊断报告 + 从题库推荐针对性练习题
- **技术参考**：pyKT (simpleKT/AKT) 知识追踪 + NeuralCD 认知诊断
- **题库来源**：CMM-Math + TAL-SCQ5K-CN 打底 + 日常作业积累 + LLM 生成变式题（CAS 验证正确性）

#### 场景 3：用户系统
- **功能**：多用户注册、年级/学科 profile、解题历史保存、学情数据隔离
- **个性化**：profile 信息自动注入到解题和诊断的 Prompt 中
- **注册时填写**：姓名、年级、重点学科
